<h4 align="center">Python ≥ 3.2.</h4>

# 

### Features

- [x] Check DA, PA
- [x] Check Moz Rank and Backlinks
- [x] Fast Checking with multi threading
- [x] Without API Moz
- [x] 100% Free
#### Requirements
- requests
- time
- CaseInsensitiveDict
- ThreadPoolExecutor

```bash
git clone https://github.com/ikballnh/domainbulk
```
```bash
cd domainbulk
```
```bash
pip install -r requirements.txt 
```
```bash
python main.py
```
